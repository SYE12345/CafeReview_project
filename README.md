## 🛠 개발환경(사용도구/언어)
#### `Back-end`
- Java
- Python

- github(형상관리)

- Gradle

#### `Front-end`
- html
- css
- js 
</br>

## 📗 핵심 기능
1. 사용자들에게 맞는 카페 목록과 상세 정보를 제공
2. 카페별로 리뷰를 편리하게 작성하고 공유할 수 있는 환경 제공
</br>

## 👩🏻‍💻 담당 역할
1. 카페 리스트 페이지 구현
2. 카페 리뷰 작성 페이지 구현
3. 회원가입/로그인시 유효성검사
</br>
